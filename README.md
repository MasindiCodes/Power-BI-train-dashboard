# Power-BI-train-dashboard
Interactive sales analytics dashboard built with Power BI using the Global Train dataset.

## Overview

This project analyzes retail sales performance using an interactive dashboard built in Power BI.

## Dataset

Global Superstore dataset sourced from Kaggle.

## Key Metrics

* Total Revenue
* Total Profit
* Total Orders
* Average Order Value

## Dashboard Features

* Revenue by region map
* Profit by product category
* Monthly sales trends
* Interactive region filters

## Tools Used

* Power BI
* Data cleaning with Power Query
* DAX measures for KPI calculations

## Files

* `global-superstore-dashboard.pbix` – Power BI project
* `superstore_dataset.csv` – dataset used
